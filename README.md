##  .env Setup

### Backend (`/backend`)

```
PORT=5000
MONGO_URI=your_mongo_uri
STEAM_API_KEY=your_steam_api_key
STEAM_API_SECRET=your_steam_api_secret
ACCESS_TOKEN_SECRET=your ACCESS_TOKEN_SECRET
<!-- NODE_ENV=development -->
```

### Frontend (`/frontend`)

```
VITE_STREAM_API_KEY=your_stream_api_key
```

##  Run the Backend

```bash
cd backend
npm install
npm run start
```

##  Run the Frontend

```bash
cd frontend
npm install
npm run dev
```
